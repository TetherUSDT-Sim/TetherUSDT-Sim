About the Project

TetherUSDT-Simulated (USDTs) is an academic and research-focused blockchain project that simulates the behavior and structure of a real-world fiat-backed stablecoin. While this token has no actual monetary backing, it is designed to mimic the properties of coins like USDT in order to provide a realistic learning and testing environment.

The project is deployed on the Binance Smart Chain (BSC) and uses the BEP-20 standard. It is intended to help students, developers, and researchers explore smart contract functionality, tokenomics, supply models, and more â€” without financial risk.

All components of this project are open-source and available for review, feedback, and contribution. The smart contract is verified on BscScan, and full documentation is available through our official whitepaper.
